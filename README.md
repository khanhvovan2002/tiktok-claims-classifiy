# tiktok-claims-classifiy
1. Preliminary investigation of the claims classification dataset with the aim of learning important relationships between variables.
2. EDA understand the impact that videos have on TikTok users
3. Conduct a hypothesis test to analyze the relationship between verified status
and video view count.
4. Built a logistic regression model that predicts verified status
5. Utilize random forest (RF) and XGBoost for enhancing performance
